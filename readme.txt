== Description ==
Adds advanced date filtering to WooCommerce order admin screen with:

* Single date filter
* Date range filter
* Preset ranges (Today, Last 7 Days, This Month)
* AJAX filtering support
* HPOS compatibility

== Installation ==
1. Upload the plugin folder to /wp-content/plugins/
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Visit WooCommerce â†’ Orders to use the filters.
